  Probability
  • P( A ) = (Number of outcomes favorable to A)/(Number of all possible outcomes of the experiment)
  • P(A) + P(A') = 1
  • If A and B are independent events then,
    P(A∩B) = P(A) × P(B)
  • If A and B are mutually exclusive events then,
    P(A∩B) = 0
  • Hence if A' denotes the complement event of A then,
    P(A∩A') = 0
  • Conditional Probability : P(A|B) denotes the probability that event A will occur given that event B has occurred already. Hence P(A|B) is given by
  • P( A | B) = P( A ∩ B ) / P( B)
    where,
    • P(A|B) = the (conditional) probability that event A will occur given that event B has occurred already.
    • P(A∩B) = the (unconditional) probability that event A and event B both occu
    • P(B) = the (unconditional) probability that event B occurs
